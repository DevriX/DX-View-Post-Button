== Description ==

A simple View Post helper button for editing and viewing Post.

From WordPress 4.4 WP developers removed "View Post" button, so this plugin can restore it upon activation.

If you like this plugin, good ratings is much appreciated.

*Note: This plugin will only work if you have existing non-responsive form.

Please visit our website for more information.

== Installation ==

Installation is very easy, download and upload via WordPress `Plugin` Section.

1. Activate it! 

2. Go to your WP Dashboard > Post > All Post > Edit (Particular post).

3. See the button is there? You're done!

== Frequently Asked Questions ==

= Where to find this button? =

It can be found beside the edit button when post edit held. Please head on to our screenshots
